Altair Documentation¶

Altair’s core functionality is to produce Vega-Lite JSON specifications, which specify mappings between data and graphical markings. These encodings can include binnings and aggregations, as well as data transformations and filtering. Beyond this, plots can be customized through a number of configuration Settings.
