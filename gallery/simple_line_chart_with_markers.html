Simple Line Chart with MarkersÂ¶

This chart shows the most basic line chart with markers, made from a dataframe with two columns.

import altair as alt
import numpy as np
import pandas as pd

x = np.arange(100)
source = pd.DataFrame({'x': x,
                     'sin(x)': np.sin(x / 5)})

alt.Chart(source).mark_line(point=True).encode(
    x='x',
    y='sin(x)'
)
