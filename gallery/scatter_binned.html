Binned Scatterplot¶

< Scatterplot | Example Gallery | Bubble Plot >

from altair import *

Chart('http://vega.github.io/vega-lite/data/movies.json').mark_point().encode(
    size='count(*):Q',
    x=X('IMDB_Rating:Q',
        bin=Bin(
            maxbins=10.0,
        ),
    ),
    y=Y('Rotten_Tomatoes_Rating:Q',
        bin=Bin(
            maxbins=10.0,
        ),
    ),
)
