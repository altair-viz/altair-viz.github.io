Github punch card visualization¶

< Horizontal Error Bars | Example Gallery | Histogram >

from altair import *

Chart('http://vega.github.io/vega-lite/data/github.csv').mark_circle().encode(
    size='sum(count):Q',
    x=X('time:T',
        timeUnit='hours',
    ),
    y=Y('time:T',
        timeUnit='day',
    ),
)
