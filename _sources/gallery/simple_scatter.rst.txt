Simple Scatter Plot
-------------------

A simple example of an interactive scatter plot using the well-known iris
dataset.


.. altair-plot::
    :code-below:

    

    import altair as alt
    from vega_datasets import data

    iris = data.iris()

    alt.Chart(iris).mark_point().encode(
        x='petalWidth',
        y='petalLength',
        color='species'
    ).interactive()
